📚 单链表实现栈
 
参照《数据结构与算法分析》教材核心思想，基于单向链表手写实现栈结构，不依赖STL容器与库函数，纯底层代码实现栈的先进后出（LIFO）特性，夯实数据结构基础。
 
✨ 项目简介
 
本项目采用单链表作为底层存储结构，实现栈的全部核心操作，完美还原教材中栈的逻辑原理，规避数组栈容量受限的问题，实现动态内存分配、按需存储，深入理解链式结构与栈的结合原理。
 
🧱 核心特性
 
- 动态内存管理，无需提前设定栈容量，无空间溢出问题
- 遵循栈先进后出原则，仅在链表表头进行入栈、出栈操作
- 纯手写底层逻辑，无任何第三方依赖，贴合教材知识点
- 代码结构清晰，注释完整，便于数据结构入门学习与课后复习
 
🎯 实现功能
 
栈初始化（创建空栈）
入栈操作（push）
出栈操作（pop）
获取栈顶元素
判断栈是否为空
遍历栈元素
清空栈、释放内存，避免内存泄漏
获取栈的有效元素个数
